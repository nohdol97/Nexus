# Nexus 콘솔 (Next.js)

이 디렉토리는 Nexus 콘솔 UI를 위한 Next.js 프로젝트입니다. `create-next-app`으로 부트스트랩되었습니다.

## 시작하기

개발 서버를 실행합니다.

```bash
npm run dev
# 또는
yarn dev
# 또는
pnpm dev
# 또는
bun dev
```

브라우저에서 http://localhost:3000 을 열어 결과를 확인하세요.

`app/page.tsx`를 수정하면 페이지가 자동으로 갱신됩니다.

이 프로젝트는 `next/font`를 사용해 Vercel의 Geist 폰트를 자동으로 최적화/로딩합니다.
